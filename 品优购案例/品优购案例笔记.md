# 品优购项目规划
## 品优购项目搭建工作
1. 项目文件夹：shopping
   +  样式类图片文件夹：images
   + 样式文件夹：css
     + CSS初始化样式文件：base.css
     + CSS公共样式文件：common.css
   + 产品类图片文件夹：upload
   + 字体类文件夹：fonts
   + 脚本文件夹：js
   + 首页文件：index.html

## 模块化开发
+ 有些样式和结构在很多页面都会出现，比如头部，底部，可以把这些结构和样式单独作为一个模块，然后重复使用。
+ 共用样式common.css：首页、列表页、详情页、版心、页面文字颜色、清除浮动
## favicon图标
favicon.ico一般用于作为缩略的网站标志，目前主要浏览器都支持favicon.ico图标。
#### 制作favicon图标
1. 把拼优购图标切成png图片
2. 把png图片转换为ico图标，需要借助于第三方网站。然后把生成的图标放到网站的根目录下。
3. html页面引入favicon图标

## 网站TDK三大标签SEO优化
SEO汉译搜索引擎优化，利用搜索引擎规则提高网站在有关搜索引擎的排名。
页面必须有三大标签来符合SEO优化：
1. title
2. description
3. keyword

## 快捷导航shortcut结构搭建
我们开始制作首页的头部和底部的时候，根据模块化开发，样式写到commom.css里面。
+ 最上部的通栏命名为shortcut，快捷导航的意思。
+ 最外层大盒子，包含版心
+ 版心包含左盒子，左浮动
+ 版心包含右盒子，右浮动
## 快捷导航shourtcut左侧开发
